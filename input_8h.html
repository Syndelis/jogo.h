◆ roda_do_mouse

roda_do_mouse
extern

Quantidade que a roda do mouse foi girada no último passo de tempo.

Positivo significa rolagem para cima, enquanto negativo significa rolagem para baixo. Note que este valor é resetado a cada passo de tempo.
